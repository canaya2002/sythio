## Page Templates

### Feature Page Template
**Purpose:** Target "[feature] AI" and "audio to [output]" keywords
**Structure:**
1. Hero: Feature name + clear value prop
2. How it works: 3-step explanation
3. Example output: Mockup showing the output
4. Use cases: Where this feature applies
5. Comparison: How Sythio does it differently
6. Related features: Cross-links
7. CTA: Start free

### Use Case Page Template
**Purpose:** Target "[use case] AI notes" and "best tool for [use case]" keywords
**Structure:**
1. Hero: Use case name + pain point
2. Problem: What happens without Sythio
3. Solution: How Sythio solves it
4. Example output: Mockup for this use case
5. Features involved: Which Sythio features apply
6. Who it's for: Audience
7. Related use cases: Cross-links
8. CTA: Start free

### Comparison Page Template
**Purpose:** Target "sythio vs [competitor]" and "[competitor] alternative" keywords
**Structure:**
1. Hero: "Sythio vs [Competitor]"
2. Quick comparison table: Key dimensions
3. Where Sythio is different: Unique advantages
4. Feature comparison: Detailed breakdown
5. Pricing comparison: If applicable
6. Who should choose Sythio: Target audience
7. CTA: Try Sythio free

### Audience Page Template

3. How Sythio helps: Audience-specific benefits

### Outcome Page Template

## Content Quality Standards

1. **No thin content** ‚Äî Every page must have genuine depth and unique value
2. **No keyword stuffing** ‚Äî Natural language, keyword-aware but not forced
3. **Concrete examples** ‚Äî Show mockups, sample outputs, real scenarios
4. **Cross-linked** ‚Äî Every page connects to at least 3 other relevant pages
5. **Distinct intent** ‚Äî No two pages target the same search intent
6. **Premium voice** ‚Äî Matches Sythio brand: calm, sharp, confident, minimal
7. **Actionable** ‚Äî Every page has a clear next step for the user
